##  Overview  
This **Password Strength Checker** is a Python-based program that analyses the strength of a password based on length, character variety, and common weak patterns.  

The program provides:
- A **score (out of 100)**  
- A **strength rating**: Weak, Moderate, or Strong  
- Helpful **tips to improve your password**

---

## Features
Checks for:
- Password **length**
- Presence of **uppercase**, **lowercase**, **numbers**, and **symbols**
- **Common weak words** 
- **Repeating characters** 


Provides:
- A numerical **strength score**
- A **strength classification**
- **Suggestions** to make passwords stronger
